# Security Policy

If you discover a security issue in Sizzle, please do not open a public issue first.

Report it privately to:

- Nebu Pookins <nebupookins@gmail.com>

Please include:

- A description of the issue
- Steps to reproduce it
- The affected version or commit, if known
- Any suggested mitigation, if you have one

I will triage the report and coordinate a fix before public disclosure when possible.
